# Code of Conduct

We're a small project. Be respectful, be specific in feedback, assume good intent.

- No harassment, no slurs, no personal attacks.
- Disagreements about technical direction are fine and welcome. Disagreements about people aren't.
- If someone's behavior is making the project a worse place to be, open an issue tagged `conduct` or email the maintainer at help@thr33wolfmoon.com.

Adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), v2.1.
